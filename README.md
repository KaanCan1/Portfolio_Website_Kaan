# Kaan Can Kurt - Kişisel Portföy Web Sitesi

Siteye gitmek için -->  https://kaancan1.github.io/Portfolio_Website_Kaan/
Merhaba!  
Bu proje benim kişisel portföy sitem. Burada hem kendimi tanıtıyorum hem de yazılım alanındaki yeteneklerimi, deneyimlerimi ve projelerimi sergiliyorum. Amacım, modern ve kullanıcı dostu bir tasarımla ziyaretçilere hakkımda hızlıca bilgi vermek ve kolayca iletişim kurmalarını sağlamak.  

## Özellikler

- **Duyarlı Tasarım**: Site hem mobilde hem de masaüstünde sorunsuz çalışıyor.  
- **Hakkımda Bölümü**: Eğitimim ve deneyimlerim hakkında detaylı bir tanıtım.  
- **Yetenekler/Deneyim**: Web ve mobil geliştirme teknolojilerim, veritabanı bilgilerim ve araçlarım interaktif bir şekilde listeleniyor. Üzerine gelince hover efektleriyle seviyeler görsel olarak belirtiliyor.  
- **Projeler Bölümü**: Şu an için yer tutucu var ama ileride yaptığım projeleri buraya ekleyeceğim.  
- **İletişim Formu**: EmailJS ile entegre. Ziyaretçiler bana doğrudan mail gönderebiliyor. Form gönderiminde başarılı/başarısız geri bildirim de veriyor.  
- **Hamburger Menü**: Mobil cihazlar için kolay navigasyon.  
- **Yukarı Çık Butonu**: Sayfada aşağı inince çıkan, tek tıkla yukarı döndürme özelliği.  
- **Sosyal Medya Linkleri**: LinkedIn ve GitHub profillerime hızlı erişim.  
- **CV İndirme**: Tek tıkla CV’ye erişim.  
- **Lottie Animasyonu**: İletişim kısmında hoş bir etkileşim katıyor.  

## Kullanılan Teknolojiler

- **HTML5** → Yapısal iskelet  
- **CSS3** → Stil ve duyarlı tasarım (medya sorguları + CSS değişkenleriyle tema yönetimi)  
- **JavaScript** → Menü, butonlar ve iletişim formu gibi etkileşimler  
- **EmailJS** → Sunucuya ihtiyaç olmadan tarayıcıdan mail gönderimi  
- **LottieFiles** → Vektör animasyonlar  
- **Font Awesome** → İkonlar  
- **Google Fonts** → "Poppins" fontu
